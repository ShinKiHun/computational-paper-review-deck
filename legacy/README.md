# legacy/

Older, pre-scroll-snap artifacts kept only for reference. **Not part of the active workflow.**

- `paper_review_template.qmd` — an earlier Quarto-based approach to the deck.
- `paper_review_template.html` — an early static HTML template.
- `styles.scss` — SCSS from the Quarto approach.
- `paper_review_style_guide.md` — an early style guide (superseded by `../DESIGN.md`).

The current, supported way to build a deck is the root `index.html` + `DESIGN.md` + `AGENTS.md`.
Safe to delete this folder if you never need the Quarto path again (`git rm -r legacy`).
