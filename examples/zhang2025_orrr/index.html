Scientific Problem

고전적 metal-atop volcano는 weak-binding 활성을 놓친다.

ExpectedModerate-binding metal이 ORR volcano 정점이어야 한다.
ObservedWeak-binding Ni/Cu/Zn도 높은 activity를 보인다.
QuestionDescriptor가 아니라 adsorption site 가정이 틀린 것은 아닐까?
Motivation
Conventional Understanding

기존 모델은 O*를 metal center 위에 붙잡아 둔다.

Atop assumptionHOO*, O*, HO*가 같은 metal center 주변에서 이어진다고 본다.
ConsequenceNi/Cu는 O* binding이 약해 HOO* → O* step에서 불리하다.
Blind spotN ligand는 coordination 환경일 뿐, reaction site로 취급되지 않는다.
Baseline model
New Idea / Hypothesis

Bridge O*가 N ligand를 active ensemble로 끌어들인다.

Site switchO*가 metal atop에서 M–N bridge site로 이동한다.
Scaling shiftNi/Cu에서 bridge O* 안정화가 weak-binding penalty를 줄인다.
InterpretationActive site는 metal atom 하나가 아니라 M–N ensemble이다.
Conceptual center
Computational Setup

계산은 geometry를 interface response와 연결한다.

01 · DFTAtop / bridge adsorption geometry를 먼저 비교한다.
02 · InterfaceElectric field와 solvation response를 site별로 분리한다.
03 · Kinetics그 차이를 pH-dependent volcano로 연결한다.
Modeling strategy
Main Result

Bridge-O*가 volcano를 weak-binding metal 쪽으로 옮긴다.

Atop pathWeak-binding metal은 HOO* → O* bottleneck으로 낮게 예측된다.
Bridge pathHigh-activity region이 Ni/Cu 쪽으로 열린다.
MeaningSite geometry가 kinetic descriptor로 작동한다.
Microkinetic core
Mechanism / Interpretation

Geometry가 field와 solvation response를 바꾼다.

FieldAtop O*와 bridge O*는 electric field에 다르게 반응한다.
Solvation용매 안정화도 site geometry에 따라 달라진다.
Mechanism그래서 pH-dependent kinetics가 site switch에 민감해진다.
Interface response
Validation / Benchmark

활성과 분광학이 같은 bridge-O* 그림을 가리킨다.

ActivitySimulated onset potential과 TOF trend가 experiment와 맞는다.
SpectroscopyN K-edge XAS와 N 1s XPS가 N–O 관련 signature를 보인다.
CautionInterface dynamics가 직접 관찰된 것은 아니다.
Validation stack
Limitations / Assumptions

약한 고리는 electrochemical interface 모델이다.

Static DFTFinite-T electrolyte에서 bridge O* accessibility는 직접 sampling되지 않았다.
InterfaceField mapping · solvation · capacitance 선택이 kinetic 예측을 흔들 수 있다.
Model gapModel catalyst는 명료하지만, 실제 SAC site는 더 disorder되어 있다.
Critical read
Discussion / Take-home

ORR은 metal-atop O*에 갇혀 있지 않다.

Scientific claimO*가 M–N bridge site로 전환되면 weak-binding에서도 활성이 살아난다.
Computational lessonAdsorption geometry는 scaling · field · solvation · kinetics를 함께 바꾼다.
Group questionAIMD/MLIP sampling으로 bridge O*가 실제 electrolyte에서 안정한지 볼 수 있을까?
